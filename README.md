# Selected Stories

An experimental web text editor with RNN/LSTM capabilities.
Everything runs in the browser using [deeplearn.js](https://github.com/PAIR-code/deeplearnjs) and [p5ml](https://github.com/ITPNYU/p5-deeplearn-js)
